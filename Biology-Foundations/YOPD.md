# YOPD/YOAD 研究视角：细胞器“故障报告”手册

> **研究背景：** 针对早发型帕金森 (YOPD) 与早发型阿尔兹海默症 (YOAD) 的 HEOR 建模。
> **核心逻辑：** 每一个宏观的临床症状（如震颤、失智），在微观上都能找到对应的“部门故障”。

## 1. 动力系统故障 (The Power Plant Failure)
| 生词 (Organelle) | 故障描述 (Pathology) | 临床/经济影响 (HEOR Impact) |
| :--- | :--- | :--- |
| **Mitochondria** | **线粒体功能障碍**。发电厂“漏电”产生氧化应激。 | **YOPD 核心诱因**。导致黑质神经元死亡，引发震颤。需评估线粒体保护剂的 ICER 价值。 |
| **ATP** | **能量货币枯竭**。细胞没有钱维持基本运作。 | 导致神经元信号中断。在模型中代表了疾病进入“不可逆期”的转折点。 |

## 2. 环卫与回收系统故障 (The Sanitation Crisis)
| 生词 (Organelle) | 故障描述 (Pathology) | 临床/经济影响 (HEOR Impact) |
| :--- | :--- | :--- |
| **Lysosome** | **溶酶体清理失效**。垃圾焚烧炉熄火，无法分解有毒蛋白。 | **YOAD/YOPD 共同点**。导致淀粉样蛋白斑块堆积。对应长期的社会护理成本（Caregiving Cost）。 |
| **Autophagy** | **自噬作用停滞**。细胞内的“自动清扫车”不动了。 | 蛋白质错误折叠的堆积点。是目前医药研发（R&D）最热门的干预路径。 |
| **Vacuole** | **存储空间过载**。在神经细胞中表现为受损物质的堆积。 | 虽然在植物中更显著，但在病理学中代表了细胞的“压力负荷”。 |

## 3. 生产与物流中断 (Manufacturing & Logistics Breakdown)
| 生词 (Organelle) | 故障描述 (Pathology) | 临床/经济影响 (HEOR Impact) |
| :--- | :--- | :--- |
| **Ribosome** | **核糖体翻译错误**。工厂接错图纸，批量生产残次品蛋白。 | **遗传性 YOPD 的源头**。涉及基因检测（Genetic Testing）在早发型患者中的经济学评价。 |
| **Endoplasmic Reticulum (ER)** | **内质网应激**。车间积压了太多折叠错误的蛋白，导致工厂炸毁（细胞凋亡）。 | 触发 **Apoptosis (凋亡)** 的关键环节。 |
| **Golgi Body** | **高尔基体分拣错误**。多巴胺等神经递质无法被正确“打包”。 | 导致突触间隙信号缺失。是左旋多巴等药物试图修复或替代的环节。 |
| **Vesicle** | **囊泡运输中断**。快递车在路上爆胎，信号无法传达给下一个神经元。 | 影响 **Exocytosis (胞吐)**。直接对应患者的运动迟缓（Bradykinesia）症状。 |

## 4. 辅助理解词汇 (Supporting Tech Terms)
* **Membrane-bound**: 膜结合的。意味着这些故障是被“隔离”在特定部门的，这为精准靶向药提供了可能。
* **Symbiosis**: 共生。线粒体曾是独立生物，这解释了为什么它有独立基因，也解释了 YOPD 遗传倾向的复杂性。
* **Pores**: 孔洞（如核孔）。是信息（mRNA）进出的唯一通道，也是某些病毒攻击的后门。

---
### 💡 学习备注：
- **重点死磕：** `Mitochondria`（动力）和 `Lysosome`（清理）。这两个词论文里会出现无数次。
- **关联逻辑：** 既然是 **Young-Onset (早发)**，意味着这些部门的故障发生得比常人早，且持续时间更长，这正是做经济学建模时“折现率”和“长期成本预测”的难点所在。
