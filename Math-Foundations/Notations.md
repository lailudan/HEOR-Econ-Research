# 🧮 数学常用符号指南 (HEOR & Economics Edition)

这份清单涵盖了从基础逻辑到高级计量经济学中常见的数学表达方式。

---

## 1. 逻辑与集合论 (Logic & Set Theory)
*这是构建模型假设（Assumptions）的底层语言。*

| 符号 | 名称 | 含义 / 例子 |
| :--- | :--- | :--- |
| $\forall$ | 全称量词 | **对于所有**。例：$\forall x \in \mathbb{R}$ (对于所有实数 x) |
| $\exists$ | 存在量词 | **存在**。例：$\exists y$ 使得 $f(y) = 0$ |
| $\neg$ | 否定 | **非**。逻辑取反 |
| $\implies$ | 蕴含 | **如果...则...**。例：$A \implies B$ (若 A 发生，则 B 必然发生) |
| $\iff$ | 充要条件 | **当且仅当**。A 与 B 逻辑等价 |
| $\in$ | 属于 | 元素属于集合。例：$i \in \{1, \dots, n\}$ (个体 i 属于样本集) |
| $\subset$ | 子集 | 集合 A 包含在集合 B 中 |
| $\cup$ | 并集 | A 与 B 的合并 |
| $\cap$ | 交集 | A 与 B 的共同部分 |
| $\mathbb{R}$ | 实数集 | 经济变量（如价格、成本）通常定义的范围 |
| $\mathbb{Z}$ | 整数集 | 离散变量（如人数、天数）的范围 |

---

## 2. 微积分 (Calculus)
*用于研究边际效应（Marginal Effects）和最优化（Optimization）。*

| 符号 | 名称 | HEOR/经济学中的含义 |
| :--- | :--- | :--- |
| $f'(x)$ | 一阶导数 | **边际变化**。如：边际成本 $MC = \frac{dC}{dQ}$ |
| $f''(x)$ | 二阶导数 | 判断极值。$f''(x) < 0$ 通常意味着效益最大化 |
| $\frac{\partial y}{\partial x}$ | 偏导数 | **保持其他变量不变**时，x 对 y 的影响 (Ceteris Paribus) |
| $\int_a^b$ | 定积分 | **曲线下面积**。在 HEOR 中常用于计算 **QALYs** (质量调整寿命年) |
| $\sum_{i=1}^n$ | 求和 | 样本汇总。例：总成本 $\sum Cost_i$ |
| $\prod$ | 连乘 | 计算联合概率或似然函数 (Likelihood) |
| $\Delta$ | 增量 | 变化量。例：$\Delta C$ (增量成本) |
| $\infty$ | 无穷大 | 极限分析或长期均衡 |

## 3. 线性代数 (Linear Algebra)
*处理多变量数据集和回归模型的矩阵语言。*

| 符号 | 名称 | 备注 |
| :--- | :--- | :--- |
| $\mathbf{v}$ | 向量 | 通常用加粗小写字母表示。一列数据（如所有患者的年龄） |
| $\mathbf{A}$ | 矩阵 | 通常用大写字母表示。整个数据集 |
| $\mathbf{A}^T$ | 转置矩阵 | 行列互换。常用于计算协方差矩阵 |
| $\mathbf{A}^{-1}$ | 逆矩阵 | 相当于矩阵的“除法”。求解线性方程组的核心 |
| $\|\mathbf{x}\|$ | 范数 | 向量的长度。常用于衡量估计误差的大小 |
| $\det(\mathbf{A})$ | 行列式 | 判断矩阵是否可逆 |
| $I$ | 单位矩阵 | 矩阵世界的“1” |

## 4. 概率与统计 (Probability & Statistics)
*HEOR 的核心：处理不确定性与数据推断。*

| 符号 | 名称 | 含义 / 例子 |
| :--- | :--- | :--- |
| $P(A)$ | 概率 | 事件 A 发生的可能性 |
| $P(A \mid B)$ | 条件概率 | **给定 B 发生的情况下**，A 发生的概率（贝叶斯核心） |
| $E[X]$ | 期望值 | **平均水平**。决策树模型中的核心指标 |
| $Var(X)$ | 方差 | 风险或波动程度 |
| $\sigma$ | 标准差 | 数据的离散程度 |
| $\mu$ | 均值 | 总体参数的平均值 |
| $X \sim N(\mu, \sigma^2)$ | 正态分布 | 变量 X 服从均值为 $\mu$，方差为 $\sigma^2$ 的正态分布 |
| $\beta, \hat{\beta}$ | 回归系数 | $\beta$ 是真实值，$\hat{\beta}$ 是我们从数据中得到的**估计值** |
| $H_0$ | 原假设 | 统计检验中那个“想要推翻”的假设 |

---

## 5. 常用希腊字母 (Greek Letters)
*在经济模型中，它们通常代表特定的参数。*

* $\alpha$ (Alpha): 常数项，截距。
* $\beta$ (Beta): 敏感度，边际影响。
* $\gamma$ (Gamma): 增长率或调节因子。
* $\epsilon$ (Epsilon): **残差/误差项**。模型无法解释的部分。
* $\theta$ (Theta): 模型中待估算的未知参数集合。
* $\lambda$ (Lambda): 拉格朗日乘数，或表示到达率。
