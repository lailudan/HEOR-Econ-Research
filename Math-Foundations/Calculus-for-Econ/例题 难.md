
# 三角极限与核心恒等式

---

## 1. 核心三角恒等式 (Must-Know)

在处理微积分极限时，这几组关系是“消去零因子”的关键。

### A. 基础母公式
* **勾股恒等式**: $\sin^2\theta + \cos^2\theta = 1$
* **正切定义**: $\tan\theta = \frac{\sin\theta}{\cos\theta}$

### B. 二倍角公式 (Double-Angle Identities)
当你看到 $\cos 2\theta$ 或 $\sin 2\theta$ 时，通常需要降幂或展开：
* $\cos 2\theta = 1 - 2\sin^2 \theta$ （最常用，用于将余弦转为正弦）
* $\cos 2\theta = \cos^2 \theta - \sin^2 \theta = (1+\sin\theta)(1-\sin\theta)$
* $\sin 2\theta = 2\sin \theta \cos \theta$

### C. 周期性与零点
* **正弦零点**: $\sin(n\pi) = 0$ ($n$ 为整数)
  * *例题回顾*: $\sin(-5\pi) = 0$
* **余弦零点**: $\cos(n\pi + \frac{\pi}{2}) = 0$

## 2. 经典题型解析

### 类型一：消去 $(1 - \cos\theta)$
**题目**: $\lim_{\theta \to 0} \frac{1 - \cos \theta}{2\sin^2 \theta}$

**解题逻辑**:
1. 利用 $\sin^2 \theta = 1 - \cos^2 \theta$。
2. 平方差展开：<!-- $1 - \cos^2 \theta = (1 - \cos\theta)(1 + \cos\theta)$ -->
   <img width="708" height="84" alt="3ab6afc3-277a-430e-9557-e6256e145639" src="https://github.com/user-attachments/assets/756ec6f1-51cc-4ffe-b2ed-f02339c003e4" />

4. **约分**:
   $$\frac{1 - \cos \theta}{2(1 - \cos\theta)(1 + \cos\theta)} = \frac{1}{2(1 + \cos\theta)}$$
5. 代入 $\theta = 0$，结果为 $1/4$。

### 类型二：处理 $\sqrt{2}$ 与倍角
**题目**: $\lim_{\theta \to -\frac{\pi}{4}} \frac{1 + \sqrt{2}\sin \theta}{\cos 2\theta}$

**解题逻辑**:
1. 观察到代入后为 $0/0$。
2. 将分母展开: $\cos 2\theta = 1 - 2\sin^2 \theta$。
3. 平方差公式: $1 - 2\sin^2 \theta = (1 + \sqrt{2}\sin\theta)(1 - \sqrt{2}\sin\theta)$。
4. **约分**并代入:
   $$\frac{1}{1 - \sqrt{2}\sin(-\frac{\pi}{4})} = \frac{1}{1 - (-1)} = \frac{1}{2}$$

## 3. 几何直观：拉格朗日恒等式 (Lagrange's Identity)

作为代数与几何的桥梁，它锁定了向量点积与叉积的关系：
$$\|\mathbf{u}\|^2 \|\mathbf{v}\|^2 - (\mathbf{u} \cdot \mathbf{v})^2 = \|\mathbf{u} \times \mathbf{v}\|^2$$

**顿悟点**:
* 该恒等式本质上是三角恒等式 $1 - \cos^2\theta = \sin^2\theta$ 的高维代数表现。
* 它证明了：**平行四边形的面积，等于两向量长度乘积的平方减去它们投影的平方。**




## 4. 二倍角公式的逻辑推导 (Logic Derivation)

不要死记硬背，记住“二倍角 = 两个相等的角相加”：

| 函数 | 和角公式起点 | 二倍角结论 |
| :--- | :--- | :--- |
| **Sine** | $\sin(A+B) = \sin A \cos B + \cos A \sin B$ | $\sin 2\theta = 2\sin\theta\cos\theta$ |
| **Cosine** | $\cos(A+B) = \cos A \cos B - \sin A \sin B$ | $\cos 2\theta = \cos^2\theta - \sin^2\theta$ |
| **变体 1** | 使用 $1 - \sin^2\theta$ 替换 $\cos^2\theta$ | $\cos 2\theta = 1 - 2\sin^2\theta$ |
| **变体 2** | 使用 $1 - \cos^2\theta$ 替换 $\sin^2\theta$ | $\cos 2\theta = 2\cos^2\theta - 1$ |

**💡 技巧：** 在微积分极限中，如果你看到 $1 - \cos 2\theta$，立刻联想到 $2\sin^2\theta$；看到 $\cos 2\theta - 1$，立刻联想到 $-2\sin^2\theta$。

### 🛡️ 极限的精确定义：$\epsilon-\delta$ 协议

- **$\epsilon$ (Epsilon)**：老板给的精度要求（y 轴）。
- **$\delta$ (Delta)**：我给出的操作范围（x 轴）。

**逻辑闭环：**
只要我能证明，对于任意小的 $\epsilon$，都能找到一个 $\delta$，使得：
当 $0 < |x - c| < \delta$ 时，必有 $|f(x) - L| < \epsilon$。
那么，极限就是铁证如山的！

**本题套路总结：**
1. 写出 $|f(x) - L| < \epsilon$。
2. 通过代数变形，把 $|x - c|$ 凑出来。
3. 观察系数，建立 $\delta$ 与 $\epsilon$ 的比例关系。
