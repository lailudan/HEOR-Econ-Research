## 🛠 维护原则 (Maintenance Principles)

1.  **DRY (Don't Repeat Yourself)**: 若多个模型共享同一逻辑内核，提取至 `Core_Logic`。
2.  **原子化**: 每篇笔记仅解决一个逻辑断点。
3.  **链接化**: 思考之间的关联通过 Markdown 内部链接实现，而非复制内容。
